## Build Setup

``` bash
$git clone https://github.com/PTaras/CRM_flask_js.git
python run app.py

result http://127.0.0.1:5000/logos
```
<h3>Возможности для пользователя:</h3>
<ul>
  <li>Ввести домен/ссылку в поле (ссылка <a href='http://127.0.0.1:5000/logos'>http://127.0.0.1:5000/add-logo</a>)</li>
  <li>Нажать кнопку «Show logo» (получить логотип)</li>
  <li>Увидеть полученный от API логотип (после нажатия кнопки «Show logo»)</li>
  <li>Сохранить или пропустить его (сохранить - «Add logo into base», пропустить - перейти на главную)</li>
  <li>Видеть сохраненные логотипы в таблице (Main или Logos)</li>
  <li>Искать записи в таблице по тексту по всем колонкам - (поле «Search» и кнопки начала и отмены поиска)</li>
  <li>Удалить логотип и запись из таблицы нажав кнопку «Delete logo»</li>
  <li>Скачать логотип нажав кнопку «Download logo»</li>
  </ul>


<p>После переходa по ссылке <a href='http://127.0.0.1:5000/logos'>http://127.0.0.1:5000/logos</a> Вы можете видеть основную структуру приложения:
<ul>
  <li>Navbar:<ul>
               <li><b>Main</b> - Главная страница</li> 
                <li><b>Logos</b> - сслылка на страницу с логотипами, по сути, это и есть главная</li>
                <li><b>Add logo</b> - ссылка на страницу с добавление логотипа или его предосмотром</li>
                <li><b>Поле поиска по странице</b> - ищем текст на странице, 2 кнопки: <i>start</i> - начало поиска, <i>esc</i> - отменить поиск или убрать найденный текст</li>
    </ul>
  <li>Таблица с логотипами</li>
  <p>Содержит колонки:
    <ul>
      <li><b>id</b> - уникальный идентификатор логотипа</li>
      <li><b>Domain</b> - название логотипа (домен)</li>
      <li><b>Logos</b> - представление логотипа в картинке</li>
      <li><b>Date_created</b> - дата добавления информации о логотипе в базу/таблицу</li>
      <li><b>Delete logo</b> - кнопка удаления информации о логотипе с таблицы и базы</li>
      <li><b>Download logo</b> - кнопка загрузки логотипа к себе на ПК</li>
  </ul>
  </p>
</ul>
